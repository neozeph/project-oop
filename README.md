# CashWizz - Expense Tracker Application

**CashWizz** is a desktop-based personal finance application developed as a second-year university project for an Object-Oriented Programming course. Built entirely in Python, it provides a simple graphical interface for users to log, manage, and review their daily expenses. 

## Key Features
* **User Authentication:** Secure registration and login system for individual users.
* **Expense Management:** Easily add, view, search, and delete expense records (including item name, date, cost, and category).
* **Spending Insights:** Automatically calculates total expenses and provides quick feedback based on spending habits.
* **Data Persistence:** Utilizes local SQLite databases to store user credentials (`loginpage.db`) and expense records (`expenseapp.db`).
* **Admin Controls:** Features to view or wipe registered users and clear all expense data.

## Technologies Used
* **Language:** Python
* **GUI Framework:** Tkinter
* **Database:** SQLite3
